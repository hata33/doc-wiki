```
   - 核心 API/概念：查看主索引中的 [[PageName]] 链接
   - 特定集成：使用 Glob 搜索（如 Glob("wiki/sources/*bedrock*")）

4. Read 原始文档获取完整内容
5. 综合信息给出答案
6. 标注所有使用的 [[APIName]]
7. 提供可运行的代码

问题：[你的问题]
```

### 查找策略

**核心 API 和概念**
```
1. 读 wiki/index/index.md
2. 找到相关的 [[PageName]] 链接
3. 跳转到对应的 wiki/api/ 或 wiki/concepts/ 页面
4. 从 Wiki 页面获取原始文档路径
5. Read 原始文档
```

**特定集成（如 Bedrock、Redis 等）**
```
1. 使用 Glob: Glob("wiki/sources/*bedrock*")
2. 或查看 wiki/index/sources_index.md
3. 找到相关页面后读取原始文档
```

**浏览某一类完整文档**
```
- 所有聊天模型：读 wiki/index/api_index.md
- 所有向量存储：读 wiki/index/api_index.md
- 所有集成：读 wiki/index/sources_index.md
```
